# Crypto Trends Landing (GitHub Pages Ready)

Сайт на чистом JavaScript, стилизованный под интерфейс CoinMarketCap и теперь работающий с **реальными данными в настоящем времени**.

## Что реализовано

- верхняя навигация и блоки метрик;
- таблица топ-50 криптовалют по капитализации;
- live-данные из CoinGecko API (`/coins/markets` + `/global`);
- поиск по названию/тикеру;
- автообновление каждые 60 секунд и кнопка `Refresh`;
- кликабельные вкладки Top/Trending/Watchlist/Prediction/Most Visited с пояснением;
- каждая вкладка применяет свой смысловой режим списка (капитализация/рост/объем/волатильность/топ-20);
- сортировка по столбцам таблицы (по возрастанию/убыванию);
- ссылки верхнего меню ведут на соответствующие страницы (Cryptocurrencies/Dashboards/Exchanges/Community).

## Локальный запуск

```bash
python3 -m http.server 4173
```

Откройте `http://localhost:4173`.

## Публикация на GitHub Pages

1. Загрузите файлы в репозиторий.
2. На GitHub: **Settings → Pages**.
3. В `Source` выберите:
   - **Deploy from a branch**
   - Branch: **main** (или текущая), folder: **/ (root)**
4. Нажмите **Save**.

Через 1–2 минуты сайт будет доступен по адресу:
`https://<username>.github.io/<repo-name>/`

## Важно

CoinGecko public API имеет лимиты запросов. Если API временно недоступно, в таблице появится сообщение об ошибке загрузки.


## API-ключ

В `script.js` добавлен ваш ключ CoinGecko demo API для более стабильной работы на GitHub Pages.

> Важно: в статическом сайте ключ виден в исходном коде браузера. Для production-безопасности лучше проксировать запросы через сервер.


По умолчанию при обновлении страницы открывается вкладка **Top** и список идет от #1 к последнему месту.

Надпись **CoinScope** в шапке снова в прежнем стиле и при клике обновляет страницу.
