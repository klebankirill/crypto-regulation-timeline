# Crypto Trends Landing (GitHub Pages Ready)

Мини-сайт на чистом JavaScript, стилизованный под интерфейс CoinMarketCap (как на вашем примере):
- верхняя навигация;
- блоки метрик;
- тематические теги;
- таблица трендовых криптовалют с динамической генерацией через `script.js`.

## Локальный запуск

Откройте `index.html` в браузере **или** запустите статический сервер:

```bash
python3 -m http.server 4173
```

и откройте `http://localhost:4173`.

## Публикация на GitHub Pages

1. Загрузите эти файлы в репозиторий.
2. На GitHub откройте: **Settings → Pages**.
3. В `Source` выберите:
   - **Deploy from a branch**
   - Branch: **main** (или текущая), folder: **/ (root)**
4. Нажмите **Save**.
5. Через 1–2 минуты сайт будет доступен по адресу формата:
   `https://<username>.github.io/<repo-name>/`

> Если хотите отдельный пользовательский сайт, используйте репозиторий `<username>.github.io`.
